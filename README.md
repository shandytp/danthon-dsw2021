# Data Science Weekends by Data Science Indonesia

Source : [Kaggle](https://www.kaggle.com/c/danthon2021/data)

### Task

Dari data *traffic* yang sudah disediakan oleh JDS, lakukan prediksi *high jam reports* di daerah tertentu, tanggal tertentu dan jam tertentu dari beberapa kota di Jawa Barat.

### Training Set

DSI dan JDS menyediakan 4 files dimana 2 file terdiri dari *reports* dan *irregularities*. Sisa file digunakan untuk labeling dan testing.

### File Descriptions

- `data_train.csv` - data training annotations
- `data_test.csv` - test set ids yang digunakan untuk prediksi
- `alerts.csv` - alerts report dari user
- `irregularities.csv` - irregularities alert dari sistem
- `sample_submission.csv` - contoh submission dengan format yang benar
